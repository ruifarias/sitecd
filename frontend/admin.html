Configuração Novidades

Defina quantos dias um artigo é considerado "Novidade".

Artigos com Data_Ult_Compra dentro deste período aparecem em "Novidades"

Colecção Actual

Define qual é a colecção "actual", usada na ordenação "Colecção" do catálogo - dentro de cada família, os artigos desta colecção aparecem primeiro, seguidos dos anos anteriores (do mais recente para o mais antigo).

Tipos de Envio

Active/desactive os tipos de envio disponíveis no checkout e edite a designação e o custo. A "Ordem" controla a posição em que aparecem no checkout (menor aparece primeiro, e é a opção pré-seleccionada).

Alertas

Artigos Reservados

Stock reservado por encomendas ainda não enviadas nem anuladas. A reserva é libertada quando a encomenda é enviada ou anulada.

A carregar artigos reservados...
Imagem Código Designação Lote Qtd. Reservada Encomenda Acção

Artigos Sem Stock mas com Imagem

Publicados, sem stock disponível, mas ainda com foto principal - candidatos a rever (despublicar ou confirmar reposição).

A carregar...
Imagem Código Designação Existência

Artigos com Stock mas sem Imagem

Publicados, com stock disponível, mas sem foto principal - a ficha aparece sem imagem no site, o que prejudica a venda.

A carregar...
Código Designação Existência

Artigos "NEW" mas sem "Ano Actual"

Marcados como Novidade, mas a colecção (Ano/Estação) não corresponde à Colecção Actual configurada em Backoffice - pode ser uma Novidade desactualizada ou uma colecção mal classificada.

A carregar...
Imagem Código Designação Colecção

Artigos com Stock Negativo nos Lotes

Lotes com saldo negativo (Disponível − Reservado < 0) - sintoma de erro de contagem/sincronização na origem.

A carregar...
Código Designação Lote Qtd. Disponível Qtd. Reservada Saldo

Artigos com Preço Custo = 0

Sem custo lançado no ano corrente (TB0001StkPrcCusto, origem) - distorce margens/relatórios.

A carregar...
Imagem Código Designação Preço Custo

Artigos sem Preço de Venda, ou com Preço de Venda = 0

Publicados sem preço válido - normalmente já despublicados automaticamente pelo sync (preço a 0€), fica aqui como rede de segurança.

A carregar...
Imagem Código Designação Preço

Artigos com IVA Incluído (is NULL)

Taxa_IVA_Incluido não definida na origem (TB0001StkArtigos) - por confirmar se o preço já inclui IVA.

A carregar...
Imagem Código Designação Último Login (origem)
